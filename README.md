# Imprenta Biobio Landing

Landing page estatica para Imprenta Biobio.

## Archivos

- `index.html`: estructura principal del sitio
- `styles.css`: estilos y responsive
- `script.js`: menu movil y navegacion activa

## Desarrollo local

Al ser una landing estatica, puedes abrir `index.html` directamente en el navegador.
